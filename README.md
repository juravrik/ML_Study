# Machine Learning #
基本的な機械学習のアルゴリズムを実装していきたい

### 概要
numpyを使用しsklearnAPIっぽい実装を目指します。

### 実行例について
sample.pyに一部クラスの実行例をまとめてあります。
--"クラス名"を実行オプションにつけることで指定ができます。
